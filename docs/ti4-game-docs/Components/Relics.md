Relics are a powerful component released in the [[Prophecy of Kings]] Expansion of Fourth Edition.

They have a wide variety of powerful abilities; some are single use, others grant passive abilities, and a select few reward victory points (with conditions).

[[File:Book of Latvinia.png|thumb]]

== Rules ==
Relics cannot be traded.

== Methods of Acquiring Relics ==

* Purging 3 matching relic fragments
* Purging 2 relic fragments using the [[The Naaz-Rokha Alliance|Naaz-Rhoka Alliance']]<nowiki/>s Fabrication faction ability
* Purging 2 relic fragments using the [[The Naaz-Rokha Alliance|Naaz-Rhoka Alliance']]<nowiki/>s promissory note
* Using the Naaz-Rhoka Alliance's hero ability
* Using the [[The Mahact Gene-Sorcerers|Mahact Gene Sorcerers]] [[Breakthroughs|Breakthrough]] ability
* Gaining a planet card from the planet deck that has a relic icon [[File:RelicSymbol.PNG|29x29px]] (these appear on the 4 planet's in [[The Fracture]])
* Being elected by the Minister of Antiquities [[Agenda Cards|Agenda Card]]
* Using the Dead World [[Exploration|Frontier Card]]
