Each player has one flagship in their [[Reinforcements]]. 

Although each flagship is represented by the same plastic model, they all have unique stats and abilities.   

Each costs 8 resources to produce, has the Sustain Damage ability and usually hits twice during combat with the majority also having one or more faction-specific abilities.
